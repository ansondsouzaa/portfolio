ABC Learning Centre

Single page application

ABC Learning Centre is a Single Page Application website developed using HTML, JavaScript, CSS, jQuery & Ajax. This websites is for an IT Training Institute which provdies IT courses. The website's purpose is to show courses and a schedule for each respective course to their students.


ABC Jobs Portal

Prototyping

This project is about creating a prototype for the 'to be made website application' of ABC Job Portal. The process of prototyping starts from writing Ideas, creating a story board, personas, selecting personas that maybe the potential users of the final Job Portal. Paper prototypes are made and then the final prototype is developed using the AXURE RP Software.


ABC Jobs Portal

Database design and development

This project is about designing and developing suitable database for the 'to be made website application' of ABC Jobs Portal. The process starts with creating a logical design using a tool called ERD Plus, this design is made for identifying entities and their attriutes . Next in line comes process like creating Logical Schema, Data flow diagram or DFD followed by a flowchart and lastly comes a Physical Schema which is created by using MySQL Workbench.


ABC Jobs Portal

Secure Website Portal

ABC Jobs is a portal enabling developers to find jobs. This portal was built using Java and Struts-2 Framework. Users are enabled to securely register into the website, they can login and can even reset forgotten password using a secure OTP method. Features like creating a profile, Searching other users and jobs by name or location are implemented.


Network Infrastructure

Networking

This project has a hypothetical scenario for building a Network Infrastructure for 30 computer devices around a small campus. A sample network architecture is made in plain paper and then on an online platform to confirm the deisgn. Once the network architecutre is finalised a Network Architecure design is made using Cisco Packet Tracer. This tools is used to verify the workability of the design and verify if the network should pass from one point to another not.
